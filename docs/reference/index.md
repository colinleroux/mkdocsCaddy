# Admonition Reference

This page provides examples of the most commonly used **MkDocs Material admonitions**.
Each example shows the rendered output followed by the Markdown required to create it.

---

# Note

!!! note
This is a note admonition. Notes are typically used for helpful information that supplements the main content.

### Markdown

```markdown
!!! note
    This is a note admonition. Notes are typically used for helpful information that supplements the main content.
```

# Collapsible Admonition

Collapsible admonitions allow readers to expand content only when needed.

??? note "Click to expand"
This is a collapsible admonition.
- item one  
- item two  
- item three

### Markdown

```markdown
??? note "Click to expand"
    This is a collapsible admonition.
```

# Multiple Paragraph Content

Admonitions can contain multiple paragraphs and lists.

!!! note
Admonitions can include multiple lines of content.

```
Example list:

- item one  
- item two  
- item three
```

### Markdown

```markdown
!!! note
    Admonitions can include multiple lines of content.

    Example list:

    - item one  
    - item two  
    - item three
```
